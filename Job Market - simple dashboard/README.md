# Data JOB Dashboard – 1st Project

## Introduction

This is my *first* Power BI project. The goal was to explore the data job market, visualize key metrics, and gain hands-on experience creating interactive dashboards.

## Skills Showcased
- Creating and customizing graphs
- Organizing dashboard pages effectively
- Developing KPI reports
- Using Power BI for data visualization and analysis

## Dashboard Overview

### PAGE 1: Job Market View
![Dashboard page 1](/Job%20Market%20-%20simple%20dashboard/Dashboard%201.png)  
This page provides an overview of the data job market. Users can compare different positions, identify high-demand roles, and analyze trends to determine which positions are currently most sought-after.

### PAGE 2: Detailed Position View
![Dashboard page 2](/Job%20Market%20-%20simple%20dashboard/Dashboard%202.png)  
This page provides a detailed analysis of individual positions. Users can explore salary ranges, required skills, and job availability, enabling informed decisions about career paths and opportunities.

## Conclusion
This project helped me develop practical skills in Power BI and create dashboards that provide clear insights into the data job market. Future enhancements could include additional KPIs, predictive analytics, and interactive features to support more advanced analysis.
