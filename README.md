# Data JOB Dashboard – 2nd Project

## Introduction
This dashboard provides insights into the job market, offering a detailed view of job trends, salaries, and required skills across different countries. The project focuses on creating an interactive and customizable experience, enabling users to explore the data in depth.

## Skills Showcased
- Implementing various filtering options, including **job title** and **country**.  
- Displaying the **percentage of total jobs per skill** for better understanding of skill demand.  
- Creating **dynamic chart metrics** and **x-axis slicers** for scalable data visualization.  
- Using **parameters** to allow users to switch between different types of data.  
- Writing **measures** to provide meaningful insights, helping users decide which job types or skills are most suitable.  
- Designing a **clean and professional dashboard layout**.  
- Adding a **useful button to reset filters** for a smoother user experience.  

## Dashboard Overview
The dashboard presents data such as job count per **job title** and **country**, allowing users to identify the most popular roles in their location. It also displays:  
- **Average salary per month and per hour** to compare pay across roles.  
- **Skills required** for different job types, including counts and averages.  
- **Job titles** alongside typical experience levels and corresponding salaries.  

## Conclusion
The goal of this project was to create a more **slick and professional design** while leveraging tools like **parameters** and **measures**. This allows the dashboard to provide **insights beyond the raw data**, helping users make informed decisions about which jobs or skills suit them best.
